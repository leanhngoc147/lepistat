Giới thiệu về Lepistat

Lepistat là một gói R mạnh mẽ được thiết kế để hỗ trợ phân tích dữ liệu thống kê trong các nghiên cứu khoa học. Gói này cung cấp nhiều chức năng hữu ích cho việc làm sạch dữ liệu, phân tích mô tả, kiểm định thống kê, và tạo biểu đồ đẹp mắt để trình bày kết quả nghiên cứu của bạn.

Được phát triển bởi các nhà nghiên cứu cho các nhà nghiên cứu, Lepistat kết hợp sức mạnh của R với sự đơn giản trong việc sử dụng, giúp rút ngắn thời gian từ dữ liệu đến những hiểu biết có giá trị.

Cài đặt Lepistat

Cài đặt từ GitHub

Bạn có thể cài đặt phiên bản mới nhất của Lepistat trực tiếp từ GitHub bằng cách sử dụng gói remotes hoặc devtools:

# Cài đặt gói remotes nếu chưa có
install.packages("remotes")

# Cài đặt Lepistat từ GitHub
remotes::install_github("leanhngoc147/lepistat")

Sử dụng lệnh trên để cài đặt trực tiếp từ repository GitHub của leanhngoc147.

Yêu cầu cài đặt

Lepistat yêu cầu:

  • R phiên bản 4.0.0 trở lên
  • Các gói phụ thuộc: dplyr, ggplot2, tidyr, stats

Các gói phụ thuộc sẽ được tự động cài đặt khi bạn cài đặt Lepistat.

Kiểm tra cài đặt

Sau khi cài đặt, bạn có thể kiểm tra xem gói đã được cài đặt thành công chưa bằng cách chạy:

library(lepistat)
lepistat_info()

Lệnh này sẽ hiển thị thông tin về phiên bản và các chức năng chính của gói.

Tính năng chính

Phân tích mô tả

Lepistat cung cấp các hàm mạnh mẽ để tạo bảng mô tả dữ liệu, phù hợp với báo cáo khoa học:

# Mô tả cấu trúc dữ liệu
des(data, gt = TRUE, vie = TRUE)

# Tạo từ điển dữ liệu
thuviendata(data)

# Tạo bảng thống kê mô tả
thongkemota(data, digits = 1, title = "Thống kê mô tả")

Báo cáo HTML tương tác

Tạo báo cáo HTML tương tác với khả năng xuất sang Word và Excel:

# Tạo báo cáo HTML tương tác
sharinganhtml(data, varlist = c("gender", "age", "bmi"), 
              title = "Báo cáo thống kê", theme = "light")

# Báo cáo có phân tầng theo biến
sharinganhtml(data, varlist = c("sbp", "chol"), by = "gender", 
              digit = 1, title = "Báo cáo theo giới tính")

Phân tích theo nhóm

Thực hiện phân tích so sánh giữa các nhóm với tính toán tỷ suất chênh (OR) và nguy cơ tương đối (RR):

# Tạo báo cáo phân tích với OR
rasengan_html(data, vars = c("disease", "blood_type"), 
              by = "gender", ratio = "OR", lang = "vie")

# Phân tích nguy cơ tương đối
rasengan_html(data, vars = c("disease"), by = "age_group", 
              ratio = "RR", title = "Phân tích nguy cơ")

Hướng dẫn sử dụng

Bắt đầu nhanh

Để bắt đầu sử dụng Lepistat, hãy tải gói và sử dụng các chức năng phân tích dữ liệu:

# Tải gói
library(lepistat)

# Tạo dữ liệu mẫu
data <- create_sample_data(n = 100)

# Xem mô tả dữ liệu
des(data)

# Xem từ điển dữ liệu
thuviendata(data)

# Tạo bảng thống kê mô tả
thongkemota(data)

# Phân tích và tạo báo cáo HTML
sharinganhtml(data = data, varlist = c("age", "gender", "bmi"))

Tạo báo cáo chuyên nghiệp

Lepistat cung cấp các hàm tạo báo cáo HTML tương tác để chia sẻ kết quả phân tích: 

# Tạo báo cáo phân tích mô tả
sharinganhtml(data, varlist = c("gender", "age", "bmi", "sbp"),
             title = "Báo cáo thống kê mô tả", digit = 1)

# Tạo bảng phân tích với nhiều tùy chọn
rasengan_html(data, vars = c("disease", "bmi"),
             by = "gender", ratio = "OR",
             title = "Phân tích theo giới tính")
